# shimi ritual ✨

אפליקציית Web מעוצבת לשגרת טיפוח אישית, בעברית RTL, מוכנה להעלאה ל-GitHub Pages.

## מה יש בפנים
- מסך בית מחודש בסגנון luxury skincare
- שגרת בוקר ושגרת ערב לפי יום
- סימון שלבים ושמירה מקומית בדפדפן
- מסך מוצרים מעוצב עם חיפוש ופילטרים
- יומן שבועי
- פרופיל קטן
- אייקון חדש + manifest + service worker כדי להרגיש כמו אפליקציה אמיתית

## העלאה ל-GitHub Pages
1. פתחי Repository חדש ב-GitHub
2. העלי את כל הקבצים והתיקיות מתוך הזיפ
3. Settings → Pages
4. Source: Deploy from a branch
5. Branch: main, Folder: /root
6. שמירה, וזהו — אחרי כמה דקות יהיה לינק.

## התקנה באייפון
פתחי את הלינק בספארי → Share → Add to Home Screen.
